# 銀行系統 (Bank System) 專案

這是一個用於模擬基本銀行業務的簡單銀行系統專案。專案旨在提供一個可擴展的基礎架構，以實現銀行業務的基本功能。

## 功能特點

- **帳戶管理：** 支援新增帳戶、查詢帳戶餘額、轉帳等功能。
- **交易記錄：** 保存每一筆交易的詳細信息，包括日期、金額、交易類型等。
- **安全性：** 實現基本的安全機制，如密碼加密和帳戶權限管理。
- **交易通知：** 通過郵件或簡訊等方式提供即時交易通知服務。
